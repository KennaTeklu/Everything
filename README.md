# Everything – 3D Knowledge Map Monolith

A single‑file container that embeds a complete 3D knowledge graph, a dashboard, and an article page inside iframes. The 3D map uses Three.js, Dexie (IndexedDB), and a custom force‑directed graph.

## 🚀 Quick Start

### Run locally with a simple HTTP server

```bash
# Python 3
python3 -m http.server 8000

# or Node.js
npx http-server . -p 8000
